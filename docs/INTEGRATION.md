# 小逆输入法 - 集成指南

## 快速开始

### 1. 克隆项目
```bash
git clone <项目地址>
cd xiaoni-ime
```

### 2. 配置讯飞 SDK

#### 2.1 注册讯飞账号
1. 访问 https://www.xfyun.cn/
2. 注册并登录
3. 创建新应用
4. 获取 AppID

#### 2.2 下载 SDK
1. 进入讯飞控制台
2. 下载"语音识别" Android SDK
3. 解压后将 `Msc.jar` 放入 `app/libs/` 目录

#### 2.3 配置 AppID
创建或编辑 `local.properties`：
```properties
IFLYTEK_APPKEY=你的AppID
```

### 3. 构建项目

```bash
# 使用 Gradle 构建
./gradlew assembleDebug

# 或直接在 Android Studio 中运行
```

### 4. 安装与启用

1. 安装 APK 到手机
2. 进入 设置 → 系统 → 语言和输入法
3. 启用"小逆输入法"
4. 点击"当前输入法"选择小逆输入法

## 使用说明

### 基础输入
- 点击字母键输入文字
- 点击"符"切换符号键盘
- 点击"⌫"删除文字
- 点击"完成"发送回车

### 语音输入
- 点击"🎤"按钮进入语音输入
- 对着手机说话
- 识别结果会自动输入
- 开启"连续模式"后可自动连续识别

### 设置
- 打开小逆输入法应用
- 调整各项设置：
  - 连续识别模式
  - 自动发送
  - 按键音效
  - 按键震动

## 注意事项

1. **首次使用需要网络**：语音识别需要联网
2. **授予录音权限**：首次语音输入时会请求权限
3. **保持麦克风畅通**：避免遮挡麦克风

## 故障排除

| 问题 | 解决方案 |
|-----|---------|
| 无法启用输入法 | 检查系统设置中是否已启用 |
| 语音输入无反应 | 检查录音权限和网络连接 |
| 识别准确率低 | 说话清晰，避免背景噪音 |
| 自动发送无效 | 检查目标应用是否支持回车发送 |
